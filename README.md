# easytier-helper
基于easytier的创建虚拟网络、加入虚拟网络，要求两者一方能访问到另一方（经典案例：其中一方有公网ip）
![alt text](doc-image/image.png)
一键命令：
```bash
curl -sSL https://raw.githubusercontent.com/My-Search/easytier-helper/refs/heads/master/easytier-helper.sh -o /tmp/easytier-helper.sh && chmod +x /tmp/easytier-helper.sh && bash /tmp/easytier-helper.sh
```
